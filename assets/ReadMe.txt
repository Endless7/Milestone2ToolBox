提示：
需要ROOT权限，设置重启生效

程序图标：
---- 手电筒（长按搜索键也可以调出）

功能菜单：
【系统调整】
---- 基带切换（需安装相关组件）
---- 更改启动画面 （在sd卡根目录下 /sdcard/m s2toolbox/bootanimations里新建目录，并将启动动画放进去，自动识别）
---- 更改系统字体 （在sd卡根目录下 /sdcard/m s2toolbox/fonts里新建目录，并将字体放进去，自动识别）
---- 更改系统型号标识 （由于超频工具MilestoneOverclock检查了型号，不识别国行ME722，所以无法超频，通过该功能可以快速修改为MotoA953）
---- 更改默认多媒体引擎 (对比见2楼)
提示：bootanimations和fonts下面的文件夹，不能包含中文和空格
【控制设置】
---- 相机键点亮屏幕
---- 语音键/中英切换键（仅摩托输入法有效）
---- 关闭屏幕底部LED灯new
---- 关闭侧滑键盘灯new
【提示音管理】new
---- 禁用相机拍照提示音
---- 禁用视频录制提示音
【通话挂断震动】new
---- 通话结束后震动提醒
---- 震动持续时间
【Wi-Fi自动管理】new
---- Wi-Fi断开后自动关闭（当Wi-Fi连接断开后，几分钟内没有新的连接，自动禁用Wi-Fi，再也不怕出门忘记关Wi-Fi了）
---- 设置延迟时间（分钟）new
【手电筒 】new
---- 打开手电筒 （因长时间使用对闪光灯不好，一次打开时间最长为2分钟，可多次打开）
---- 设置手电筒亮度级别new
---- 设置最长点亮时间new
【高级工具】
---- sd2rom hack（需安装相关组件）
【快捷】new
---- 快速重启手机
---- 重启到恢复模式
