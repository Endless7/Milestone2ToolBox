【KyleTang原创】里程碑2工具箱 1.6.0 (3/2更新，机锋首发)

提示：
需要ROOT权限，带星号的设置项，重启后生效。

PS：有些地方是直接修改的系统，所以卸载本软件或wipe仍然生效，所以也可以认为本工具是一个ROM修改工具。

程序菜单快捷方式：
---- 手电筒（开启/关闭手电筒，手电筒参数在工具箱里设置；长按搜索键，可以调用手电筒）
---- 锁屏（锁屏，长按搜索键，可以调用）
---- 电源（快速关机/快速重启/重启到恢复模式/系统电源选项，长按搜索键，可以调用）

功能列表：
【系统调整】
---- 基带切换（需安装相关组件）
---- 更改启动画面 （在sd卡根目录下 /sdcard/m s2toolbox/bootanimations里新建目录，并将启动动画放进去，自动识别）
---- 更改系统字体 （在sd卡根目录下 /sdcard/m s2toolbox/fonts里新建目录，并将字体放进去，自动识别）
提示：bootanimations和fonts下面的文件夹，不能包含中文和空格
---- 更改系统型号标识 （由于超频工具MilestoneOverclock检查了型号，不识别国行ME722，所以无法超频，通过该功能可以快速修改为MotoA953）
---- 更改默认多媒体引擎 (对比见2楼)
---- 更多设置
-------【按键管理】
----------- 相机键点亮屏幕
----------- 语音键/中英切换键（仅摩托输入法有效）
-------【背景灯管理】
----------- 关闭屏幕底部LED灯
----------- 关闭侧滑键盘灯
-------【消除提示音】
----------- 禁用相机拍照提示音（仅禁用拍照声音，暂时没法禁用对焦声音）
----------- 禁用视频录制提示音
【增强功能】
---- 通话震动提醒
-------【拨号接通后震动】
----------- 拨号接通后震动提醒
----------- 震动持续时间
-------【通话结束后震动】
----------- 通话结束后震动提醒
----------- 震动持续时间
-------【45秒震动】
----------- 通话中45秒震动提醒
----------- 震动持续时间
---- Wi-Fi自动关闭
------- Wi-Fi断开后自动关闭（当Wi-Fi连接断开后，几分钟内没有新的连接，自动禁用Wi-Fi，再也不怕出门忘记关Wi-Fi了）
------- 设置延迟时间（分钟）
---- 最小内存管理（解决了MinFreeManager开机应用不生效的问题）
【附加工具】
---- 手电筒功能
------- 打开手电筒 （因长时间使用对闪光灯不好，一次打开时间最长为2分钟，可多次打开）
------- 设置手电筒亮度级别
------- 设置最长点亮时间
---- 电源功能
------- 快速重启手机
------- 重启到恢复模式
【高级工具】
---- sd2rom hack（需安装相关组件）
